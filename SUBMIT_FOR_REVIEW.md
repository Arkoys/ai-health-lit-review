# Submit a paper or resource for ad hoc review

You can send any of the following directly in chat:

## 1) Paper URL
Example:
- https://pubmed.ncbi.nlm.nih.gov/41935405/
- https://arxiv.org/abs/2501.12345

## 2) PDF or screenshot
Send the file and optionally add context:
- what claim you want checked
- what section matters most
- what decision depends on it

## 3) A specific claim / fact to audit
Example:
- "Est-ce vrai que les scribes IA réduisent le burnout clinique ?"
- "Peux-tu vérifier si cette étude montre vraiment un gain causal ?"

## 4) A bundle of sources
Send:
- links
- notes
- screenshots
- abstracts
- press articles

## Best prompt format
Use this template:

Sujet / question:
Contexte:
Ce que tu veux que je vérifie:
Niveau de profondeur voulu: rapide | normal | approfondi
Sources jointes:

## What Hermes will do
Depending on the input, Hermes can:
- summarize the paper/resource
- extract methods, results, limits, and gaps
- check whether the conclusion matches the evidence
- compare multiple sources
- identify weak claims, confounders, or overstatements
- produce a clean markdown note for archiving

## Suggested output modes
Ask for one of these:
- "résumé rapide"
- "critique méthodologique"
- "fact-check"
- "comparaison de 2-5 papiers"
- "note markdown prête à archiver"
