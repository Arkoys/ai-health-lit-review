% TITLE GUIDANCE
%
% The title is a concise scientific claim.
% It should state:
%   (1) WHAT was done,
%   (2) ON WHAT population, data, or setting,
%   (3) FOR WHAT purpose or outcome.
%
% A good title is:
% - precise and informative (not aspirational),
% - understandable without the abstract,
% - falsifiable in principle (the implied claim could be wrong),
% - aligned with the actual scope of the study.
%
% Avoid hype words (e.g. "novel", "transformative", "groundbreaking").
% Use a colon only when it adds clarity.
%
% Ask yourself:
% - Could a reviewer reasonably say "this is not supported by the results"?
% % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % %
