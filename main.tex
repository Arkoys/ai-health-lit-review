% TITLE GUIDANCE
%
% The title is a concise scientific claim.
% It should state:
%   (1) WHAT was done,
%   (2) ON WHAT population, data, or setting,
%   (3) FOR WHAT purpose or outcome.
%
% A good title is:
% - precise and informative (not aspirational),
% - understandable without the abstract,
% - falsifiable in principle (the implied claim could be wrong),
% - aligned with the actual scope of the study.
%
% Avoid hype words (e.g. "novel", "transformative", "groundbreaking").
% Use a colon only when it adds clarity.
%
% Ask yourself:
% - Could a reviewer reasonably say “this is not supported by the results”?

% Authorship for all LiGHT theses and publications follows the established
% ethical authorship standards of ICMJE
%
% Authorship is not a matter of hierarchy or status. It reflects contribution.
%
% Roles and order:
% - First author:
%   The individual who made the primary intellectual contribution and led the
%   execution and writing of the work.
%
% - Middle authors:
%   Contributors who made substantial intellectual, methodological, or analytic
%   contributions. Ordering should reflect relative contribution unless authors
%   agree otherwise.
%
% - Joint first or joint senior authorship:
%   Permissible when contributions are genuinely equal. This must be explicitly
%   agreed upon and documented in advance.
%
% - Last author:
%   Typically the person with overall scientific responsibility for the work
%   (e.g. principal supervisor or PI), accountable for its integrity.
%
% Authorship order must be discussed explicitly and cannot be pre-agreed. 
% It should be revisited as the project evolves based on contributions, and finalized before submission.
%
